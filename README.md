
# 🎓 Guide Universitaire Congo

Guide Universitaire Congo est une plateforme d’orientation destinée aux nouveaux bacheliers du Congo-Brazzaville.

## 🌍 Objectif du site
Aider les étudiants à :
- Trouver les universités au Congo
- Découvrir les filières disponibles
- S’orienter selon leur profil
- Accéder à des informations utiles pour leurs études

## 📚 Fonctionnalités
- Liste des universités
- Liste des filières
- Système de recherche
- Espace d’orientation
- Section Premium (services avancés)
- Interface mobile optimisée

## 💳 Services Premium
Certaines fonctionnalités sont payantes, comme :
- Vérification des résultats du bac (250 FCFA)
- Orientation avancée

## ⚙️ Technologies utilisées
- HTML
- CSS
- JavaScript

## 📱 Accessibilité
Le site est conçu pour être utilisé facilement sur téléphone Android et ordinateur.

## 🚀 Auteur
Projet développé pour améliorer l’orientation universitaire au Congo-Brazzaville.
